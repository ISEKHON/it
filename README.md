## ğŸ“Š Function Cheat Sheet (Quick Reference)

```excel
# Basic Math
=SUM(A1:A10)              # Add all values
=AVERAGE(B1:B5)           # Calculate mean
=MAX(C1:C20)              # Largest value
=MIN(C1:C20)              # Smallest value

# Logical
=IF(A1>50, "Pass", "Fail")                    # Simple condition
=IF(A1>=90, "A", IF(A1>=80, "B", "C"))       # Nested IF

# Lookup
=VLOOKUP("P001", A1:B10, 2, FALSE)           # Vertical lookup
=HLOOKUP("Jan", A1:M5, 3, FALSE)             # Horizontal lookup

# Counting
=COUNT(A1:A100)           # Count numbers
=COUNTA(A1:A100)          # Count non-empty
=COUNTBLANK(A1:A100)      # Count empty
=COUNTIF(B1:B50, ">100")  # Count with condition

# Text
=CONCATENATE(A1, " ", B1) # Join text (or use &)
=UPPER(A1)                # Convert to uppercase
=LOWER(A1)                # Convert to lowercase
=LEN(A1)                  # Text length

# Date/Time
=TODAY()                  # Current date
=NOW()                    # Current date & time
```

## ğŸ“ Important Comparisons (Tabular Format)

### Hardware vs Software
| Hardware | Software |
|----------|----------|
| Physical components | Programs/instructions |
| Can touch | Cannot touch |
| Examples: CPU, RAM, HDD | Examples: Windows, MS Office |

### RAM vs ROM
| RAM | ROM |
|-----|-----|
| Volatile (loses data) | Non-volatile (permanent) |
| Temporary storage | Permanent storage |
| Faster | Slower |
| Expensive | Cheaper |

### HDD vs SSD
| HDD | SSD |
|-----|-----|
| Mechanical (moving parts) | Electronic (no moving parts) |
| Slow (150 MB/s) | Fast (3500 MB/s) |
| Cheap (â‚¹3/GB) | Expensive (â‚¹40/GB) |
| Noisy | Silent |
| Fragile | Durable |

### Inkjet vs Laser
| Inkjet | Laser |
|--------|-------|
| Liquid ink | Toner powder |
| Slow (5-15 ppm) | Fast (20-40 ppm) |
| Best for photos | Best for text |
| Cheap printer | Expensive printer |
| Expensive per page | Cheap per page |

### Internet vs Intranet
| Internet | Intranet |
|----------|----------|
| Global network | Organization's private network |
| Public access | Restricted access |
| Owned by no one | Owned by organization |
| Example: www.google.com | Example: Company portal |
